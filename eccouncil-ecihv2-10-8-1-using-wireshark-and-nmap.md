## Using Wireshark and NMAP


### Objectives:

At the end of this episode, I will be able to:

Understand what the concept of collecting and analyzing information by using the
Wireshark tool entails.

Explain why using the Wireshark and NMAP tools should be a part of the collecting
and analyzing information steps taken by the IH&R team.

Identify how to install and use the Wireshark and NMAP tools.
